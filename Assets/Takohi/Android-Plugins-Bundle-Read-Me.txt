/////////////////////////////////////////////////////////////////////////

Android Vibration Plugin

/////////////////////////////////////////////////////////////////////////

Notes

This plugin only works on a Android device.

An AndroidManifest.xml has been added in order to add the required permission VIBRATE in case you want to add vibration to your notification.
If you want to use your own AndroidManifest.xml, please add this line in it:
<uses-permission android:name="android.permission.VIBRATE”/>

-------------------------------------------------------------------------

Examples

in  [Project] /Takohi/Examples/AndroidVibrationPlugin you will find one example.

-------------------------------------------------------------------------

Demo

You can download a demo to run on your Android device here:
http://www.takohi.com/data/unity/plugins/android_vibration/unity_android_vibration_plugin_demo.apk

/////////////////////////////////////////////////////////////////////////

Android Speech Recognition Plugin

////////////////////////////////////////////////////////////////////////

Notes

This plugin only works on a Android device.

An AndroidManifest.xml has been added in order to add the required permission RECORD_AUDIO.
If you want to use your own AndroidManifest.xml, please add this line in it:
<uses-permission android:name="android.permission.RECORD_AUDIO"/>

For better performance, Internet is also required.
